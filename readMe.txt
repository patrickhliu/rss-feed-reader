************
Rss Reader
************
The idea for this project comes from:
	Trying to make a simple clone of feedly / google reader.

Project Details:
	Create a RSS feed reader

Hosted Link:
	http://176.32.230.9/pliurss.com/

Instructions:
	Enter a rss feed url into the search bar and press the "get feed" button.
	All articles in the feed will display.
	By default, 5 articles per page are shown.
	This can be adjusted by user (max is 50 articles per page).

Example feeds:
	Freakonomics  - http://feeds.feedburner.com/freakonomicsradio?format=xml
	Marc Maron    - http://www.wtfpod.com/podcast/rss
	Adam Carolla  - http://theadamcarollashow.libsyn.com/rss
	Joe Rogan     - http://joeroganexp.joerogan.libsynpro.com/rss
	Bill Simmons  - http://espn.go.com/espnradio/feeds/rss/podcast.xml?id=2864045
	USA Today     - http://rssfeeds.usatoday.com/usatoday-NewsTopStories
	NPR Fresh Air - http://www.npr.org/rss/rss.php?id=13
	NPR This American Life - http://feeds.thisamericanlife.org/talpodcast?format=xml
